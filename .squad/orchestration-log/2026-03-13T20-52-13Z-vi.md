### Implementation Details

1. **Created scripts/unix_dependency_plan.py:**
   - Classifies dependencies into apt-managed and pip-managed groups
   - Apt-managed packages: python3-pygame, python3-websockets, python3-mutagen
   - Pip-managed: all others, preserving requirement specifiers
   - Single source of truth to prevent drift between install and update paths

2. **Updated install.sh:**
   - Detects Raspberry Pi via /proc/device-tree/model
   - Applies apt-first strategy for known Raspberry Pi OS packages
   - Falls back to pip for PyPI-only dependencies

3. **Updated update.sh:**
   - Uses same dependency plan
   - Performs apt installs only when root or passwordless sudo available
   - Otherwise logs warning and falls back to pip for unattended cron-style updates

4. **Release safety:**
   - Helper must be explicitly packaged in release ZIP (validated by scripts/package_release.py whitelist)
