## Key Design Decisions

1. **Frame-seeking:** Inclusive comparison (`position >= timestamp`) for intuitive playback behavior
2. **Recording API:** Auto-generated filenames with timestamp-based naming
3. **File management:** All operations scoped to configured directory for safety
4. **Status structure:** Standardized dict with file, position_ms, duration_ms, is_playing, state
5. **Exception handling:** Use Python built-in exceptions (FileNotFoundError, FileExistsError, ValueError, RuntimeError)
